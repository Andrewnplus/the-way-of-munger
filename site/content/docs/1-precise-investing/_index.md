# 精準投資

- 巴菲特與蒙格的的過人之處：能從小資產中擠出資金，投資到別處
- 國家稅率若遠高於其他國家，會付出很大代價（公司們選擇離開）
- 有資本主義的創造性破壞，才有生產力的提高
- 別因錯過好機會而沮喪，這是無法避免的投資過程
- 年紀變大後，蒙格投資以優秀的管理層和公司為目標
- 聰明又勤奮的人總有辦法賺大錢，這點永遠不會改變
- 投資要非常有耐心，等到好機會出現時才買進
- 易解決的事蒙格馬上做，否則事情只會越積越多
- 遇到必須處理的難題時，巴菲特與蒙格會反覆琢磨問題，想辦法解決
- 幸運的秘訣不是雄心壯志，而是降低期望，對於尋找伴侶尤其如此
