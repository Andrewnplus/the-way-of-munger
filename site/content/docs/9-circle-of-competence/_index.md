# 能力圈

- 有人覺得加密貨幣很先進，但這些不過是對敲詐、綁架、逃稅有用的東西
- 價值投資永不過時，以較低價格買進較高價值，這是投資的本質
- 過度分散沒好處，找到四個好機會就已足夠
- 在人性這阻力面前，很多事很難做好
- 人成功後易鬆懈，這是人性使然
- 蒙格從未退場觀望，他總能在所尋機會中選擇最好的投資
- 你的投資樣貌很大程度取決於自身能力
- 現今年輕人想出人頭地、致富，須比過往付出更多努力
- 驅動世界發展的不是貪婪，而是嫉妒
- 現在的世界很奇怪，生活變好，幸福感卻變低
- 做好每日工作，理智地面對困難，就能成功
- 期望要符合現實，坦然接受生活的苦與甜
