# 專注

- 若不在市場競爭中充分發揮財富優勢，那麼有錢也沒有意義
- 服務業只能全力為客戶消除所有痛點，才能超越其他對手
- 蒙格成功靠的不是智力，而是專注力
- 想清楚，找到答案，然後行動
- 所有成功文明必然具備完善的社會安全網
- 遍地賭場是文明之恥
- 要成功的必要兩件事：不做愚蠢的事、活得久
- 越成功的機構，越易充斥冗員，這是人性
- 蒙格不願槓桿操作：他對背負巨額債務的資產表很反感，他對人性總存戒心
- 致富是個令人享受的過程，慢慢致富才能細細體會其中的美妙滋味
- 投資時該對可能的大災難有心理準備，其他時候當個愚蠢的樂觀主義者就行
- 逆境來臨時該迎難而上，這才是積極向上的態度
