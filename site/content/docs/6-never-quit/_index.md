# 永不退場

- 能分辨權威人士說話的對錯，才算接受過良好教育
- 遇到問題徹底理解它，問題就解決一半。搞懂後才能更好地解決問題
- 只要努力工作，準確找到幾個機會，這些少數機會就夠了
- 投資的秘訣，在於你能看懂那屬於你，別然看不懂的大機會
- 若騙子知道老實人賺多少錢，就不會當騙子了
- 蒙格能成功不是因為善於解題，而是善於遠離難題，只找簡單的事做
- 蒙格投資追求的是永遠不必退場
- 蒙格想徵的人是那些自知極限在哪，不會不自量力的人
- 人都想找到能幫忙致富的基金經理人，但是好的經理人非常難找
- 蒙格的長壽秘訣：不對人性期待過高
- 公司沒有官僚風氣，上層管理者又頭腦清楚，能為投資者帶來巨大優勢
- 蒙格對於自身成就感到幸運，但不覺驕傲
- 遇到難纏的問題或是人，解法是敬而遠之
