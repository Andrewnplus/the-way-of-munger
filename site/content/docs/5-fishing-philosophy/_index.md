# 釣魚哲學

- 優質基金經理人的特徵：遇到全符合者，值得託付所有資金給他
  - 絕對誠實
  - 談起投資哲學時，邏輯清晰、胸有成竹
  - 收費公平合理
  - 所涉領域，參與者少
  - 年輕，有很長時間能長線投資
- 釣魚的秘訣有二：在有魚處釣魚；記住前一條規則
- 若將心理學的理論與其他學科結合，就能大幅提升思維能力
- 多數時候該做的就是做好眼前事，然後順其自然
- 尋找伴侶時，找個對你有較低期望的人
- 管理工作最易犯的錯誤：知道該換人但遲遲不下決心
- 面對災難不需太悲觀，它總會過去
- 夠精明，小筆資金就能得到良好投資報酬率。資金越龐大，難度越高
