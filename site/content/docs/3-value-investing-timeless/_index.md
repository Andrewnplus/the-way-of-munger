# 價值投資永不過時

- 到了老年若有許多值得回味的成就，這樣的人生很有意義
- 閱讀的好處之一是能與歷史上最優秀的人交流
- 人一生都有兩三個很好的投資機會，關鍵是能否掌握
- 價值投資的基本道理永不過時，就像數學原理一樣有效
- 無整合能力，就不可能正確認識現實世界
- 先學專業化思維，再藉跨學科思維正確認識世界
- 持續追求智慧、等待時機，這種做法永遠不會過時
- 大量閱讀和思考的好處是若做好它們，就不太需要做其他事情
- 追求理智是人生的最大樂趣
  - 你能幫自己和別人遠離災禍，成為有用的人
  - 清楚須避開哪些事，也戒除對你沒好處的缺點
- 給年輕人的建議：好好做人、講誠信、有道德，生活會更簡單，工作會更順利
- 歷史和現實來看，追求絕對平等的國家猶如死水，放棄絕對平等的國家反而欣欣向榮
