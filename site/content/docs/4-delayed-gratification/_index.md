# 延遲滿足

- 懂得延遲滿足，就能經營好公司
- 人要做感興趣的事，不喜歡的事再怎麼逼自己都不會做好
- 最有意義的事莫過於建立家庭，養兒育女
- 指數投資出現後，主動管理的基金管理人有 95%無法跑贏它
- 經理人管理資金越大，實現的報酬率就越低
- 若誠實正直，認真學習，奉行延遲滿足，最後必為贏家
- 身教勝於言教，要子女變好，自己先變好
- 領悟太慢的原因，往往是錯誤思想盤繞腦海，無法擺脫
- 遇到好機會，一定要敢下重注
- 若做好長期投資的準備（30–40 年）、僅須持有最多 3 檔股票並持有 3–4 年就能達標
- 投資屬於長期，你得承受住 50%的跌幅而面不改色
- 別試圖躲過股市大跌，若沒遇過代表你不夠積極
- 無論做什麼，只要不斷學習，磨練技能，自然會做更好
- 要與優秀者共事，你得先成為優秀的人
- 世界不變的是「難」：
  - 失敗無法避免，也才能體會克服困難的喜悅
  - 積極地面對人生，在有限中活出自身價值
- 真正愛你的人是一同面對困難、克服難關的人
- 人生就是一連串逆境，每次都讓我們能變更好
