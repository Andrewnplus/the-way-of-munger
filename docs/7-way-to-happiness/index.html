                      幸福之道 #

                      • 尋找自己擅長、具優勢的公司，才能做好投資
                      • 別隨波逐流，那不會有好結果
                      • 降低預期，為自己的小願望努力，就會過得更幸福
                      • 遇到問題，習慣逆向思考，就能更理智地解決問題
                      • 比特幣純粹是反社會的東西，社會需要的是真實貨幣
                      • 特斯拉的銷量上升是因為馬斯克讓人們相信他能治好癌症
                      • 一個文明社會要健康發展，需要的是有效的激勵機制
